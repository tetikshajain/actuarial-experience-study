## Actuarial Mortality Experience Analysis
# Project Overview

This project analyzes Canadian mortality data using actuarial life tables. The objective is to simulate how insurance companies analyze mortality risk and evaluate expected versus actual death experience within a life insurance portfolio.

Using official Canadian life table data, the project builds a mortality dataset, explores demographic patterns, and applies actuarial methods to estimate expected mortality outcomes.

# Key Goals
- Clean and structure actuarial life table data
- Explore mortality patterns across age and sex
- Simulate a life insurance portfolio
- Estimate expected deaths using mortality rates
- Compare expected vs actual mortality experience

# Methods Used
- Data cleaning with Python and pandas
- Exploratory data analysis and visualization
- Mortality modeling using life table probabilities
- Expected vs Actual (A/E) mortality analysis

# Technologies
- Python (pandas, numpy, seaborn, matplotlib)
- Jupyter Notebook
- SQL
- Excel

# Dataset
Canadian life table data containing:
- mortality probabilities
- survival counts
- life expectancy estimates
These metrics form the foundation of actuarial modeling in life insurance and pension risk analysis.

# Applications

The methods demonstrated in this project reflect common actuarial tasks performed in insurance companies and reinsurance firms, including mortality analysis, experience studies, and risk modeling.
